# SPO2 算法说明

## 1. 参数定义（唯一来源）
- 采样率：`SPO2_SAMPLE_RATE = 100Hz`
- 算法窗口：`SPO2_ALGO_WINDOW_S = 5s`
- 算法窗口点数：`SPO2_ALGO_WINDOW_SIZE = 500`
- 环形缓冲区：`SPO2_RING_BUFFER_S = 8s`
- 缓冲区点数：`WAVE_BUF_SIZE = 800`

说明：
- `SPO2_ALGO_WINDOW_SIZE` 用于每次计算时取最近多少点参与算法。
- `WAVE_BUF_SIZE` 仅用于存储容量，不等于计算窗口。
- 两者不重复：一个是“算多长”，一个是“存多少”。

## 2. 数据链路
1. F310 发送文本帧：`red,ir\r\n` (定宽 4 位数字格式)
2. F303 在 UART1 异步解析出原始的 `red_val/ir_val`
3. 解析值进入 `SPO2_Algo_PushData(red, ir, &f_red, &f_ir)`：
   - 首先经过一阶 IIR 低通滤波器滤除高频噪声（毛刺）
   - 滤波后的数据 `f_red/f_ir` 存入环形缓冲区
   - 滤波后的数据 `f_red/f_ir` 输出给 UI 进行自适应动态缩放绘制
   - 滤波后的数据 `f_red/f_ir` 转发给电脑上位机进行显示
4. 当累计到 500 新点（5s）触发一次 `SPO2_Algo_Process()`
5. `SPO2_Algo_GetResult()` 输出 `SpO2/HR/PI/R/AC`

## 3. 核心计算流程
1. 先统计最近 5 秒 IR 窗口的 AC 摆幅（`max-min`）
2. 若 `AC > 200`，直接判定为 dropout，丢弃本段窗口，不更新算法结果
3. 若窗口正常，进行心率周期估计（周期法）：
   - 一阶 IIR 低通预滤波（`a=0.125`），降低噪声同时减小相位延迟
   - 自适应峰值检测（IR）：采用滞回+显著性+不应期的状态机
     - 上阈值 `thr_up = min + 0.6*(max-min)`、下阈值 `thr_down = min + 0.5*(max-min)`
     - 峰显著性 `prominence >= 0.15*(max-min)`，抑制重搏波/肩峰误检
     - 不应期 ≈ `SPO2_SAMPLE_RATE/4`（约 250ms），避免同周期内重复计峰
   - 周期估计：取最近 3 个峰间隔的中位数作为周期，再换算心率
4. 以心率周期为窗，提取红光/红外的 `max/min`
5. 计算 `AC/DC`，得到 `R = (AC_red/DC_red)/(AC_ir/DC_ir)`
6. 由 `R` 映射 `SpO2`，并更新 `PI`
7. 对结果做历史平滑并置更新标志

## 4. Dropout 判定与处理
- 判定对象：最近 5 秒窗口内的 IR 数据
- 判定方式：计算窗口内 `AC = max-min`
- 判定条件：`AC > 200` 判为 dropout
- 处理策略：判为异常后丢弃本窗口，等待下一个 5 秒窗口
- 对外提示：发送纯文本 `ALGO_SEGMENT_DROPPED`
- 约束：提示文本不得包含数字，避免被上位机解析为波形点
